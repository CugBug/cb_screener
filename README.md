# 可转债避雷筛选系统

半自动可转债筛选管线，基于 AkShare 数据源，通过四级管线（数据获取 → 粗筛 → 排除项 → 财务评分）输出风险评级报告。

## 筛选逻辑

### 一级粗筛（3条规则）

| 条件 | 规则 | 预期淘汰率 |
|:---|:---|:---|
| 价格区间 | 80 < 现价 < 120 | ~80% |
| 评级下限 | 排除评级低于 A（如 C, CC, B, BB, BBB 等） | ~5% |
| 溢价率预警 | 溢价率 > 200% 且价格 > 110 标注预警（不排除） | — |

### 一键排除项（5条硬性标准）

| 序号 | 排除标准 | 判断逻辑 |
|:---|:---|:---|
| ① | 正股价格 < 5元 | `正股价 < 5.0` |
| ② | 连续2年以上亏损 | 最近2年年报 `净利润` 均为负 |
| ③ | 剩余期限<6个月 + 溢价率>20% | `剩余年限 < 0.5 and 溢价率 > 20%` |
| ④ | 多次拒绝下修 | 标注"需人工核查"（无法自动获取） |
| ⑤ | 信用评级 < A | 排除 BBB+ 及以下 |

> **触犯 ≥ 2 条：直接标记为排除，建议回避**

### 财务指标评分（8项）

| 指标 | 数据来源 | 安全(2分) | 预警(1分) | 高危(0分) |
|:---|:---|:---|:---|:---|
| ① 货币资金/流动负债 | 资产负债表 | > 80% | 50-80% | < 50% |
| ② 资产负债率 | 财务摘要 | < 60% | 60-70% | > 70% |
| ③ 应收账款/营收 | 资产负债表+财务摘要 | < 20% | 20-50% | > 50% |
| ④ 经营现金流/净利润 | 财务摘要（每股经营现金流/每股收益） | > 1 | 0.5-1 | < 0.5 |
| ⑤ 毛利率 | 财务摘要 | > 20% | 10-20% | < 10% |
| ⑥ 连续亏损时长 | 财务摘要（净利润逐年检查） | 无亏损 | 部分亏损 | 连亏2年 |
| ⑦ 审计意见 | 默认标准无保留（需人工核查） | — | — | — |
| ⑧ 股权质押比例 | 质押数据 | < 30% | 30-50% | > 50% |

### 评分计算公式

每项指标按阈值判定为 **3 级**：

| 级别 | 得分 | 含义 |
|:---|:---|:---|
| 绿色（安全） | 2 分 | 指标在安全阈值内 |
| 黄色（预警） | 1 分 | 指标在预警区间 |
| 红色（高危） | 0 分 | 指标超出预警阈值 |

**财务总分** = 加权求和 / 满分 × 100：

```
财务总分 = (Σ(指标得分ᵢ × 权重ᵢ) / (2 × Σ权重ᵢ)) × 100
```

其中 8 项权重之和为 1.0（满分 2 分 × 1.0 = 2），即：

```
财务总分 = Σ(指标得分ᵢ × 权重ᵢ) × 50
```

**示例**：某转债 3 项绿色(2分)、4 项黄色(1分)、1 项红色(0分)：

| 指标 | 得分 | 权重 | 加权 |
|:---|:---|:---|:---|
| 货币资金/流动负债 | 2 | 0.10 | 0.20 |
| 资产负债率 | 1 | 0.15 | 0.15 |
| 应收账款/营收 | 1 | 0.10 | 0.10 |
| 经营现金流/净利润 | 2 | 0.15 | 0.30 |
| 毛利率 | 0 | 0.10 | 0.00 |
| 连续亏损 | 1 | 0.15 | 0.15 |
| 审计意见 | 2 | 0.10 | 0.20 |
| 股权质押 | 1 | 0.15 | 0.15 |
| **合计** | — | 1.00 | **1.25** |

**财务总分 = 1.25 × 50 = 62.5**

### 风险等级判定

```
if 触犯排除项 ≥ 2 条:
    风险等级 = [排除]（直接跳过，不再看财务评分）
else:
    if 财务总分 ≥ 80: 风险等级 = [安全]
    elif 财务总分 ≥ 60: 风险等级 = [关注]
    elif 财务总分 ≥ 40: 风险等级 = [谨慎]
    else: 风险等级 = [回避]
```

> **注意**：触犯 ≥ 2 条排除项的转债，无论财务评分多少，一律标记为 **[排除]**。

## 数据接口

| 数据 | AkShare 接口 | 频率 |
|:---|:---|:---|
| 转债实时行情 | `bond_zh_hs_cov_spot()` | 每日 |
| 转债基本信息（溢价率/评级/正股价等） | `bond_zh_cov()` | 每日 |
| 正股财务摘要（净利润/毛利率/资产负债率等） | `stock_financial_abstract_ths(symbol)` | 季报后 |
| 资产负债表（货币资金/应收账款/流动负债） | `stock_financial_debt_ths(symbol)` | 季报后 |
| 股权质押比例 | `stock_gpzy_pledge_ratio_em()` | 每周 |
| 到期税前收益率 | `bond_cb_jsl()` (集思录) | 每日 |

## 运行方式

### 环境准备

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 命令行

```bash
# 完整流程（粗筛 + 排除项 + 财务评分），输出 Excel
python main.py --mode full --output excel

# 仅粗筛，输出控制台候选列表
python main.py --mode rough --output console

# 完整流程，同时输出 Excel 和控制台
python main.py --mode full --output both

# 指定转债分析（逗号分隔）
python main.py --mode detail --codes 127061,113632
```

### 配置参数

编辑 `config.py` 可调整所有筛选阈值和评分权重：

```python
# 粗筛
ROUGH_PRICE_MIN = 80        # 最低价格
ROUGH_PRICE_MAX = 120       # 最高价格
ROUGH_RATING_MIN = "A"      # 最低评级

# 排除项
STOCK_PRICE_MIN = 5.0       # 正股最低价
CONSECUTIVE_LOSS_YEARS = 2  # 连续亏损年数
NEAR_EXPIRY_YEARS = 0.5     # 临近到期阈值(年)
PREMIUM_HIGH = 20.0         # 高溢价率阈值

# 财务评分权重
WEIGHT_CASH_TO_CURRENT_LIAB = 0.10
WEIGHT_DEBT_RATIO = 0.15
WEIGHT_CFO_TO_NET_PROFIT = 0.15
# ... 详见 config.py
```

## 输出

- **Excel 报告** (`output/筛选报告_YYYYMMDD.xlsx`)：概览页 + 详情页
- **控制台报告**：概览表格 + 排除债券详情 + 低分债券详情


## 局限性

| 局限 | 说明 |
|:---|:---|
| 下修记录 | "多次拒绝下修"无法自动获取，标记为"需人工核查" |
| 审计意见 | 财务摘要不含审计意见字段，默认标注"标准无保留" |
| 数据时效 | 财务数据基于最新财报，季报发布有延迟 |
| 剩余年限 | 基于上市时间 + 6年估算，非精确到期日 |
| 到期收益率 | 来自集思录 `bond_cb_jsl()`，未登录默认仅返回 30 只，其余标 N/A |
