# 可转债避雷筛选系统

半自动可转债筛选管线，基于 AkShare 数据源，通过五级管线（数据获取 → 粗筛 → 排除项 → 财务评分 → 投资推荐）输出 Excel 和控制台报告。

## 筛选逻辑

### 一级粗筛（3条规则）

| 条件 | 规则 | 预期淘汰率 |
|:---|:---|:---|
| 价格区间 | 80 < 现价 < 120 | ~80% |
| 评级下限 | 排除评级低于 A（如 C, CC, B, BB, BBB 等） | ~5% |
| 溢价率预警 | 溢价率 > 200% 且价格 > 110 标注预警（不排除） | — |

### 一键排除项（5条硬性标准）

| 序号 | 排除标准 | 判断逻辑 |
|:---|:---|:---|
| ① | 正股价格 < 5元 | `正股价 < 5.0` |
| ② | 连续2年以上亏损 | 最近2年年报 `净利润` 均为负 |
| ③ | 剩余期限<6个月 + 溢价率>20% | `剩余年限 < 0.5 and 溢价率 > 20%` |
| ④ | 多次拒绝下修 | 标注"需人工核查"（无法自动获取） |
| ⑤ | 信用评级 < A | 排除 BBB+ 及以下 |

> **触犯 ≥ 2 条：直接标记为排除，建议回避**

### 财务指标评分（8项）

| 指标 | 数据来源 | 安全(2分) | 预警(1分) | 高危(0分) |
|:---|:---|:---|:---|:---|
| ① 货币资金/流动负债 | 资产负债表 | > 80% | 50-80% | < 50% |
| ② 资产负债率 | 财务摘要 | < 60% | 60-70% | > 70% |
| ③ 应收账款/营收 | 资产负债表+财务摘要 | < 20% | 20-50% | > 50% |
| ④ 经营现金流/净利润 | 财务摘要（每股经营现金流/每股收益） | > 1 | 0.5-1 | < 0.5 |
| ⑤ 毛利率 | 财务摘要 | > 20% | 10-20% | < 10% |
| ⑥ 连续亏损时长 | 财务摘要（净利润逐年检查） | 无亏损 | 部分亏损 | 连亏2年 |
| ⑦ 审计意见 | 默认标准无保留（需人工核查） | — | — | — |
| ⑧ 股权质押比例 | 质押数据 | < 30% | 30-50% | > 50% |

每项指标 0/1/2 分，加权求和换算为百分制：

```
财务总分 = Σ(指标得分ᵢ × 权重ᵢ) × 50
```

示例：3 绿(2) + 4 黄(1) + 1 红(0)，权重加权和 1.25 → 总分 62.5

### 风险等级

```
触犯 ≥ 2 排除项 → [排除]
财务总分 ≥ 80    → [安全]
财务总分 ≥ 60    → [关注]
财务总分 ≥ 40    → [谨慎]
否则            → [回避]
```

### 投资推荐评分

在排除雷区的基础上，对候选转债进行"该不该买"的投资评级。

#### 公式

```
推荐分 = 估值分 × 0.5 + 质量分 × 0.5
```

#### 估值分（满分 100）

| 子项 | 权重 | 评分逻辑 |
|:---|:---|:---|
| 纯债价值比（现价/纯债价值） | 60% | <0.85 → 满分，0.85-0.95 → 67%，<1.0 → 33%，>1.0 → 0 |
| 溢价率 | 40% | <20% → 满分，20-50% → 60%，50-100% → 25%，>100% → 0 |

#### 质量分（满分 100）

| 子项 | 权重 | 评分逻辑 |
|:---|:---|:---|
| 财务总分 | 70% | 直接复用财务指标评分 × 0.7 |
| 评级加成 | 30% | AAA→30, AA+→24, AA→18, AA-→12, A+→6, A→0 |
| 排除扣分 | 惩罚 | 触犯 1 条 → -25 分 |

#### 推荐等级

| 推荐分 | 等级 | 说明 |
|:---|:---|:---|
| ≥ 80 | ★★★★★ 强烈推荐 | 估值低 + 基本面健康 |
| 60-80 | ★★★★ 推荐关注 | 整体不错，可重点关注 |
| 40-60 | ★★★ 可考虑 | 有亮点但存在短板 |
| 20-40 | ★★ 暂不考虑 | 性价比不高 |
| < 20 / 排除≥2条 | ★ 回避 | 有硬伤或严重高估 |

**剩余年限闸门**：剩余 < 4 个月的转债，推荐等级上限锁死 ★★。

## 数据接口

| 数据 | AkShare 接口 | 频率 | 调用时机 |
|:---|:---|:---|:---|
| 转债实时行情 | `bond_zh_hs_cov_spot()` | 每日 | Step 0 全量 |
| 转债基本信息（溢价率/评级/正股价） | `bond_zh_cov()` | 每日 | Step 0 全量 |
| 纯债价值 | `bond_zh_cov_value_analysis(code)` | 每日 | Step 1.5 逐只 |
| 剩余年限（精确） | `bond_cb_summary_sina(code)` | 实时 | Step 1.5 逐只 |
| 正股财务摘要 | `stock_financial_abstract_ths(symbol)` | 季报后 | Step 1.5 逐只 |
| 资产负债表 | `stock_financial_debt_ths(symbol)` | 季报后 | Step 1.5 逐只 |
| 股权质押比例 | `stock_gpzy_pledge_ratio_em()` | 每周 | Step 1.5 全量 |

## 运行方式

### 环境准备

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

### 命令行

```bash
# 完整流程（粗筛 + 排除项 + 财务评分 + 投资推荐），输出 Excel
.venv\Scripts\python main.py --mode full --output excel

# 仅粗筛
.venv\Scripts\python main.py --mode rough --output console

# 完整流程 + 控制台输出
.venv\Scripts\python main.py --mode full --output both

# 指定转债分析
.venv\Scripts\python main.py --mode detail --codes 127061,113632

# 完整流程 + 额外关注指定转债（无论是否通过粗筛都会纳入分析）
.venv\Scripts\python main.py --mode full --output excel --codes 127061,127056,118031
```

### 配置参数

编辑 `config.py`：

```python
# 粗筛
ROUGH_PRICE_MIN = 80        # 最低价格
ROUGH_PRICE_MAX = 120       # 最高价格
ROUGH_RATING_MIN = "A"      # 最低评级

# 排除项
STOCK_PRICE_MIN = 5.0       # 正股最低价
CONSECUTIVE_LOSS_YEARS = 2  # 连续亏损年数
NEAR_EXPIRY_YEARS = 0.5     # 临近到期阈值(年)

# 财务评分权重
WEIGHT_DEBT_RATIO = 0.15
WEIGHT_GROSS_MARGIN = 0.10
# ... 详见 config.py

# 投资推荐分
INVEST_WEIGHT_VALUATION = 0.5   # 估值权重
INVEST_WEIGHT_QUALITY = 0.5     # 质量权重
```

## 输出

Excel 报告 (`output/筛选报告_YYYYMMDD_HHMMSS.xlsx`)：

| 列 | 说明 |
|:---|:---|
| 转债代码 / 转债名称 | 基本信息 |
| 现价 / 溢价率 / 纯债价值 | 估值数据 |
| 评级 | 信用评级 |
| 排除项计数 / 财务总分 / 风险等级 / 风险建议 | 风险排查结果 |
| 推荐分 / 推荐等级 / 推荐原因 | 投资建议及理由 |

## 项目结构

```
cb_screener/
├── config.py              # 可配置参数（阈值、权重、等级）
├── data_fetcher.py        # AkShare API 封装 + 安全数值解析
├── rough_filter.py        # 一级粗筛（价格 + 评级 + 溢价率）
├── exclusion_check.py     # 5条排除项检查
├── financial_scorer.py    # 8项财务指标加权评分
├── investment_scorer.py   # 投资推荐评分（估值 + 质量）
├── report_generator.py    # Excel/控制台报告输出
├── main.py                # CLI 主入口 + 错误处理 + 中间结果保存
├── requirements.txt       # 依赖（akshare, pandas, openpyxl）
└── output/                # 报告输出目录
```

## 局限性

| 局限 | 说明 |
|:---|:---|
| 下修记录 | "多次拒绝下修"无法自动获取，标记为"需人工核查" |
| 审计意见 | 财务摘要不含审计意见字段，默认标注"标准无保留" |
| 数据时效 | 财务数据基于最新财报，季报发布有延迟 |
| 纯债价值 | 来自东方财富可转债价值分析，单只 API 调用增加耗时 |
| 剩余年限 | 来自新浪财经债券概况，单只 API 调用增加耗时 |
| 实时行情 | 每次运行现价/溢价率不同，推荐结果随市场变化 |
